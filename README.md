# chat bot
- streamlit을 사용하여 웹에서 사용가능한 chatbot을 구현한다.
## 기능
1. 답변 생성 : 사용자 질문에 대해 문서를 참고하여 답변을 생성한다.
2. 새로운 정보 저장 : 사용자가 새로운 정보를 입력하고 DB에 저장한다.
3. 모델 응답 저장 : 모델이 생성한 답변을 DB에 저장한다.
4. DB 체크 : DB에 저장된 정보를 확인한다.
5. 데이터 삭제 : DB에 저장된 데이터를 제거한다.
